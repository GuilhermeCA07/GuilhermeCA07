## 📌 Projetos em destaque

### 🔀 Balanceamento de carga em redes SDN

Ambiente experimental desenvolvido com **Mininet, Ryu e OpenFlow** para estudar e comparar estratégias de balanceamento de carga. O projeto avalia algoritmos como Round-Robin e uma abordagem adaptativa baseada em métricas da rede.

### 👥 Sistema Banco de Talentos

Aplicação web construída em **PHP, MySQL, HTML, CSS e JavaScript**, seguindo o padrão MVC. O sistema gerencia candidatos, vagas, entrevistas, candidaturas e processos de contratação, com filtros e paginação.

### 📊 Monitoramento de redes

Laboratórios de monitoramento com **SNMP e Cacti**, coleta de métricas de tráfego, criação de gráficos e implementação de estratégias de fallback para manter a observabilidade dos dispositivos.

### 🛡️ Segurança com IDS/IPS

Ambiente de testes com **Snort 3, Wazuh, Linux e Wireshark** para criação de regras, inspeção de pacotes, detecção de ameaças e análise de eventos de segurança.

### 📡 Automação e IoT

Projetos utilizando **ESP8266 e Arduino**, com sensores, LEDs, buzzer, conexão Wi-Fi e integração com APIs para envio de alertas.
